# Progetto Pasticceria

## Come avviare il progetto

1. Installa le dipendenze nel frontend:
   cd pasticceria-frontend
   npm install

2. Avvia il server React:
  npm start

3. Avvia JSON Server nel backend:
  cd pasticceria-backend
  json-server --watch db.json --port 3001
